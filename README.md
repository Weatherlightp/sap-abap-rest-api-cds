# SAP ABAP REST Countries Integration

## Overview

Personal SAP S/4HANA project developed using ABAP Development Tools (ADT) in Eclipse.

The goal of this project is to learn how to do a complete end-to-end SAP ABAP application by:

- Consuming data from the REST Countries API
- Processing JSON responses
- Persisting data into custom database tables
- Exposing the data through CDS Views
